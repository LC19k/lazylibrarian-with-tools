# 📋 Kobo Device Setup Checklist  
Use this checklist when setting up a new Kobo device for use with:

- LazyLibrarian  
- Calibre  
- Calibre‑Web  
- kepubify  
- NickelDBus  
- Optional KOReader  

---

# 🧰 1. Initial Kobo Setup

### ✔ Complete Kobo onboarding  
- Connect to Wi‑Fi  
- Sign in to Kobo account  
- Download latest firmware  

### ✔ Disable unnecessary features (optional)  
- Pocket  
- OverDrive  
- Recommendations  

---

# 🔧 2. Install NickelMenu

### ✔ Download NickelMenu  
https://github.com/pgaskin/NickelMenu

### ✔ Copy to `.kobo/`  
### ✔ Reboot device  

---

# 🔄 3. Install NickelDBus (Recommended)

### ✔ Download `nickeldbus.kobo`  
### ✔ Copy to `.kobo/`  
### ✔ Add NickelMenu entries  
### ✔ Reboot  

---

# 📡 4. Configure Sync Options

### ✔ Add “Sync (Kobo Cloud)”  
### ✔ Add “Sync (Calibre‑Web)”  
### ✔ Add NickelDBus sync entries  

---

# 📚 5. Configure Calibre

### ✔ Enable KoboTouchExtended  
### ✔ Enable:  
[✔] Set series information

[✔] Set series index

[✔] Upload covers

[✔] Generate thumbnails

[✔] Use KEPUB format

---

# 🔌 6. USB Sync Test

### ✔ Connect Kobo  
### ✔ Verify device appears in Calibre  
### ✔ Send a test book  
### ✔ Verify:  
- Series metadata  
- Cover thumbnail  
- KEPUB format  

---

# 📡 7. Calibre‑Web Sync Test

### ✔ Tap “Sync (Calibre‑Web)”  
### ✔ Verify book downloads  
### ✔ Verify metadata  

---

# 🧪 8. Reading Position Sync Test

### ✔ Open book  
### ✔ Read a few pages  
### ✔ Tap “Sync Reading Position”  
### ✔ Verify Calibre‑Web updates  

---

# ✔ Summary

This checklist ensures:

- Perfect metadata  
- Perfect covers  
- Working Wi‑Fi sync  
- Working USB sync  
- Reading stats sync  
- Optional KOReader support
